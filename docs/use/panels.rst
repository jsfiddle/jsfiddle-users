=============
Custom Panels
=============

.. figure:: /_static/screenshots/custompanels.png
   :align: right
   :figwidth: 222px

One can change the language of a panel. To do so, one needs to open 
the Panels Section in the Sidebar on the left.

Languages currently supported:

* HTML

* CSS
* SCSS

* JavaScript
* Coffeescript
* JavaScript 1.7
